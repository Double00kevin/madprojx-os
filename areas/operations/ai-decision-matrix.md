## When to Use Each Tool

### Claude (Cowork + Code)
**Use for:** Doing things. Building things. Creating things.

- Drafting LinkedIn posts, blog articles, email outreach
- Creating documents (proposals, reports, presentations)
- Building and deploying website code (Claude Code on KITT)
- Running scheduled tasks (daily news digest, weekly content brief)
- Managing Cloudflare infrastructure via MCP connector
- Drafting and searching email via Gmail connector
- Multi-step workflows that require file creation or editing

**Don't use for:** Research that needs citations. Long documents over 100K tokens. Real-time social media intelligence.

### Perplexity Pro
**Use for:** Finding things. Verifying things.

- Competitor research with cited sources
- Market sizing and industry statistics
- Fact-checking blog articles before publishing
- Finding current pricing, features, or capabilities of AI tools
- Deep Research mode for comprehensive reports on a topic
- Any question where "according to what source?" matters

**Don't use for:** Creating content, writing code, or executing multi-step tasks.

### Gemini 2.5 Pro
**Use for:** Processing massive inputs.

- Analyzing a client's 100+ page operations manual to find AI opportunities
- Reviewing large codebases or documentation sets
- Processing long meeting transcripts or interview recordings
- Any task where the input is too large for Claude's context window
- Multimodal analysis (images, video, audio alongside text)

**Don't use for:** Short tasks, content creation, or anything that fits comfortably in Claude's context window. Don't use as a default — use when the input size demands it.

### SuperGrok
**Use for:** Understanding what's happening on X right now.

- Finding trending conversations about AI + small business
- Social listening — what are small business owners saying about AI?
- Identifying X accounts and threads to engage with
- Researching what content formats and topics are resonating on X
- Real-time sentiment around AI tools, vendors, or news

**Don't use for:** Anything that doesn't involve X/Twitter data. Not a general research tool.

## Quarterly Audit

Every 3 months, review each subscription:

1. How many times did I use this tool in the last 30 days?
2. Could I have done the same task with a tool I'm already paying for?
3. Is the monthly cost justified by the value I got?

If a tool isn't being used at least weekly, consider downgrading or canceling. Subscriptions are overhead until proven otherwise.
